# Movie Theater Site on JS

В этом учебном проекте я работаю с готовой версткой. Я удаляю рекламные блоки, меняю жанр отображаемого фильма, изменяю задний фон фильма,
динамически формирую список фильмов на странице и сортирую их, добавляю им нумерацию. Также я работаю с формой, отправляя данные в объект,
обрезаю названия фильмов, которые длиннее 21 символа, удаляю фильм из списка по нажатию на корзину.
