# Movie Theater Site on JS

В этом учебном проекте я работаю с готовой версткой сайта. Удалил рекламные блоки, поменял жанр отображаемого фильма, изменил задний фон
фильма, динамически сформировал список фильмов на странице и отсортировал их, добавил им нумерацию. Также я работал с формой, отправляя 
данные в объект, обрезал названия фильмов, которые длиннее 21 символа и удалил фильм из списка по нажатию на корзину.

В проекте я использовал DOM для работы с элементами на странице, обработку событий, работу с массивами. Применил функциональный 
подход.
